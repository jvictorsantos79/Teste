# Tela de Cadastro

Exemplo simples de cadastro utilizando Flask.

## Requisitos

- Python 3.11+
- Flask

Instale as dependências com:

```bash
pip install -r requirements.txt
```

## Execução

Para iniciar a aplicação:

```bash
python app.py
```

Acesse `http://localhost:5000` para visualizar a tela de cadastro.
